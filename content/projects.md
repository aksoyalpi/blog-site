### AI-Powered Documentation API for Care Homes

Built an AI API that accelerates patient documentation in care homes through voice transcription. Care workers record spoken notes, and the system transcribes and structures them into proper patient records — covering diseases, treatments, and daily observations. This significantly reduces documentation time and improves record quality for staff.

Sole developer — handled the full lifecycle from requirement engineering through deployment.

**Tech stack:** FastAPI, Celery, RabbitMQ, PostgreSQL, Docker
